# 규칙

1. 본인이 일주일 동안 풀 문제 개수 아래 테이블에 작성

|은성|인하|창현|
|---|---|---|
|?|?|5

  - 당연히 더 풀어도 상관없음
  - 5문제로 정했으면 월화수목금 한 문제씩 해도 되고 하루에 5문제 풀어도 됨(뭐가 더 좋은지는 본인이 알거라 생각)

2. 인증 방식
  - 개인 브랜치 만들고 거기에 본인이 풀었던 문제 기록
  - 개인 브랜치는 자기 성 영어로 작성 ex) choi, ko, jeon
  - 커밋 메시지 양식
    - 백준을 푼 경우: 'BOJ 1111 clear'
    - 기타 다른 사이트: '누구나 알아볼 수 있게 작성' ex) 프로그래머스를 풀었을 때 'programmers 1234 clear'

3. 타인의 브랜치에 절대 커밋 금지

4. 사용 언어: 자유

5. 푸는 방식: 자유
  - ai를 돌려도 되고 블로그 글을 복붙해도 됨(자기손해임)
